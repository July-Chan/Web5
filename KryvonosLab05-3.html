YouTube

YouTube - найпопулярніший сервіс для перегляду відео.

Wikipedia

Wikipedia - онлайн-енциклопедія, що наповнюється користувачами.

Instagram - соціальна мережа для обміну фотографіями та відео.

GitHub - платформа для хостингу проєктів та спільної розробки програмного забезпечення.

Pinterest

Pinterest - сервіс для обміну натхненням у вигляді фотографій та відео.

Facebook

Facebook - соціальна мережа для спілкування та обміну інформацією.
